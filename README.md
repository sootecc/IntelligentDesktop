# 📦 Intelligent Desktop

지저분한 바탕화면, 이제 **지능형 박스**로 깔끔하게 정리하세요!
Intelligent Desktop은 바탕화면의 아이콘들을 종류별로 상자(Box)에 담아 관리할 수 있게 해주는 프로그램입니다.

## 🚀 시작하기

1. **실행**: `IntelligentDesktop.UI.exe`를 더블 클릭하여 실행합니다.
   - (처음 실행 시 빈 박스가 하나 생성됩니다.)
2. **설치 (선택 사항)**: `install.bat`를 실행하면 바탕화면에 바로가기를 만들고 시작 메뉴에 등록할 수 있습니다.

## ✨ 주요 기능

### 1. 🗃️ 스마트 박스 (Box)
- 파일이나 폴더를 드래그하여 박스에 넣을 수 있습니다.
- 박스를 닫거나 프로그램을 종료하면, **파일들이 자동으로 바탕화면으로 돌아옵니다.** (안심하세요!)
- **박스 생성**: 트레이 아이콘 우클릭 -> `새 박스 만들기` 또는 기존 박스 메뉴 -> `새 박스`.
- **박스 설정**: 상단 제목을 더블 클릭하여 이름 변경, 톱니바퀴 아이콘으로 색상/투명도 변경.

### 2. 🕒 시계 위젯
- 트레이 아이콘 우클릭 -> `시계 위젯 켜기/끄기`.
- 박스 근처로 가져가면 **착! 하고 달라붙는 자석 효과**가 있습니다.
- 우클릭하여 **배경색**과 **투명도**를 내 맘대로 꾸밀 수 있습니다.

### 3. 💾 안심 종료
- 프로그램을 종료하려면 작업 표시줄 오른쪽 아래(트레이)의 **파란색 큐브 아이콘**을 우클릭하고 `종료`를 누르세요.
- 종료 시 **모든 파일은 바탕화면의 원래 위치로 안전하게 복구**됩니다.
- 윈도우를 그냥 꺼도 알아서 파일들을 뱉어내고 종료됩니다.

## 🛠️ 문제 해결

- **파일이 안 보여요!**: 트레이 아이콘 우클릭 -> `모든 박스 보이기`를 눌러보세요.
- **프로그램을 지우고 싶어요**: `uninstall.bat`를 실행하면 깨끗하게 삭제됩니다.
- **설정 초기화**: `%AppData%\IntelligentDesktop` 폴더를 삭제하면 초기 상태로 돌아갑니다.
